# Termennetwerk configuratie

Voorbeeld configuratie van het Termennetwerk gebruikt tijdens de focusweek om de Erfgeo bestanden (Gemeentegeschiedenis, Verdwenen dorpen en Departementen) doorzoekbaar te maken.

Om de voorbeelden in een lokale Termennetwerk instantie zichtbaar te maken worden de bestanden in `datasets` en `queries` in deze gekopieerd naar lokale Termennetwerk structuur onder `packages/catalog`. Zie [Termennetwerk documentatie](https://github.com/netwerk-digitaal-erfgoed/network-of-terms/tree/master/packages/graphql) voor meer informatie voor het draaien van een lokale versie. 

De queries maken gebruik van de data die op de [RCE Linked Data omgeving](https://linkeddata.cultureelerfgoed.nl/rce/histgeo) staat.
